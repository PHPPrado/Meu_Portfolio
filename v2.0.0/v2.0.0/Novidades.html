Projeto com banco de dados

Criei um sistema de gerenciamento de banco de dados utilizando Java e mySQL, o maior objetivo deste projeto era praticar POO e CRUD, este assim sendo meu primeiro projeto com banco de dados, você pode conferir um vídeo de seu funcionamento na aba projetos na tela inicial.

Funcionalidades

  • Sistema de login - O usuário é capaz de entrar no sistma com seu email e senha, caso ele não tenha uma conta registrada, o sistema cria um novo registro;

  • Validação de senha - O sistema de validação acontece com uma busca pelo banco de dados, onde a senha fornecida é comparada a senha presente no banco e relacionada aquele usuário;

  • Adicionar novos livros - O usuário é sim capaz de adicionar novos livros a biblioteca, informando seu título e autor;

  • Sistema de busca - Existem 2 sistemas de busca, o por autor e por título, onde a diferença é apenas no critério de busca, mas em ambos os casos os sistema busca os dados relacionados a pesquisa e os retorna trazendo todas as informações necessárias para o usuário, como por exemplo a disponiblidade dos livros;

  • Reservar e devolver livros - Basta digitar o nome do livro e caso o mesmo esteja disponível, ele será adicionado ao inventario do usuário, feito isso o sistema lista aquele livro como não mais disponível caso outro usuário faça login, o contrário acontece no caso de uma devolução.
